# Student Analyzer - Unit Test

## Mô tả
Chương trình phân tích điểm học sinh.

## Chức năng

- Đếm số học sinh đạt loại Giỏi (>= 8)
- Tính điểm trung bình hợp lệ (0-10)

## Công nghệ

Java  
JUnit 5  
Maven

## Chạy chương trình

Run test bằng IntelliJ hoặc:

mvn test
